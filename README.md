# Plant-Database-System (Demo)

This project is a Java console application connected to a SQL database that manages plants.

## How to Run
1. Create a local database using `sample.sql`.
2. Populate the database with any plant information.
3. Run the main class (`Plants_app.java`).

## Main Functionality
- `Database.java`: handles database connections, queries, and updates.

## Skills Demonstrated
- Java programming
- SQL database connectivity and queries
